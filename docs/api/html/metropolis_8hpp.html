Detailed Description

Metropolis algorithm for Euclidean path integral sampling.

This class performs Monte Carlo updates of a lattice configuration according to the probability distribution 

P[x] ∝ exp(−S_E[x])

where S_E is the discretized Euclidean action of the double well model.

The algorithm ensures detailed balance and ergodicity.
