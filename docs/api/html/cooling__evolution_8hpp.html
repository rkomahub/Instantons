Cooling analysis and instanton density measurement.

Cooling suppresses short-distance fluctuations, revealing semiclassical instanton structure.

Measure instanton density during cooling evolution.

Starting from a Monte Carlo configuration, the routine performs cooling sweeps and records:

  • instanton count
  • instanton density
  • action per instanton

Output format: 

n_cool, n_inst, density, action, s_per_inst
