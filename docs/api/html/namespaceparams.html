◆ a

constexpr double params::a = 0.05
constexpr

Lattice spacing in Euclidean time.

Must be small compared to the oscillation time scale: tau_osc ~ (4 eta)^{-1}

◆ dx_width

constexpr double params::dx_width = 0.5
constexpr

Width of Gaussian proposal distribution.

Trial updates are generated as 

x_i → x_i + N(0, dx_width)

Should be tuned to obtain ~50% acceptance rate.

◆ eta

constexpr double params::eta = 1.4
constexpr

Position of classical minima of the double well potential.

The potential is 

V(x) = (x^2 - eta^2)^2

The classical instanton action is 

S0 = 4 eta^3 / 3

◆ hot_start

constexpr bool params::hot_start = true
constexpr

Initial configuration selector.

true → hot start (random in [-eta, eta]) false → cold start (x_i = +eta)

◆ N

constexpr int params::N = 800
constexpr

Number of lattice points in Euclidean time.

The total Euclidean time extent is 

beta = N * a

In order to resolve instanton physics, one requires a << tau_osc beta >> tau_tunnel
