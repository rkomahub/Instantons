Construct a lattice configuration.

Parameters
NNumber of lattice points.
etaParameter defining the minima of the double well potential.
hot_startIf true, initialize randomly in [-eta, eta]. If false, initialize in the classical vacuum x = +eta.
genRandom number generator (injected).

The hot start corresponds to a disordered configuration, while the cold start corresponds to a classical minimum.
