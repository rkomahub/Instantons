◆ step()

void Metropolis::step ( )

Perform one full Metropolis sweep.

Perform one Metropolis sweep over all lattice sites.

For each site i, the algorithm proposes 

x_i → x_i + δx ,   δx ~ N(0, dx_width)

and accepts with probability 

min(1, exp(−ΔS))

where ΔS is the change in the local Euclidean action terms that depend on x_i.
