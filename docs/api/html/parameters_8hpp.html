Detailed Description

Global simulation parameters for the Euclidean lattice instanton simulation.

The system is the quantum mechanical double well potential 

V(x) = (x^2 - eta^2)^2

discretized on a one-dimensional Euclidean lattice of size N with lattice spacing a and periodic boundary conditions.
